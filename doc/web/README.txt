The 'doc/web/' directory is a shadow directory for the main '/web'
directory. The files in it contain documentation on how to use the files
in the main directory. This document gives a brief overview.

The '/web' directory contains the following files, which simply serve as
mockups for early development:

day.css
day.html
    Day view.

month.css
month.html
    Month view.

week.css
week.html
    Week view.

year.css
year.html
    Year view.
